REQUIREMENTS
------------

The minimum requirement by this project template that your Web server supports PHP 5.4.0.


INSTALLATION
------------

### Composer

~~~
composer install
~~~

Migrate database with dummy data

~~~
php yii migrate
~~~

run the project

~~~
php yii serve --port=8888 
~~~

for unit testing run

~~~
vendor/bin/codecept run unit    
~~~
